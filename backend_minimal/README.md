# 🚀 Minimal Backend - AI Smart Grabber

## 📦 Установка и запуск

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск сервера
uvicorn main:app --host 0.0.0.0 --port 5000
```

## 🔗 Проверка в браузере

- http://172.20.10.2:5000/
- http://172.20.10.2:5000/check  
- http://172.20.10.2:5000/test

## 🦾 Использование ESP32-Servo

```
ESP32 → GET http://172.20.10.2:5000/check
Ответ:
ALLOW  → хватать
DENY   → игнорировать
```

## 🌑 Обучение модели астероидов

### 1. Подготовка датасета
```bash
# Создайте папку с изображениями астероидов
mkdir asteroid_images
# Добавьте туда фотографии астероидов (.jpg, .png, .bmp)
```

### 2. Обучение модели
```bash
python train_asteroids.py
```

### 3. Использование с обученной моделью
```bash
# Запуск с поддержкой вашей модели астероидов
uvicorn main_with_yolo:app --host 0.0.0.0 --port 5000
```

## 🔥 Важно

- ESP32-CAM НЕ ТРОГАЕМ, используем /capture
- Backend — единственный мозг
- ESP32-Servo — тупо исполнитель
- Ваша модель астероидов автоматически интегрируется в систему
